# トップページLP: ルール正確化 + 役職カード画像 設計

## 目的

トップページ（`frontend/src/pages/index.tsx`）の文言が、実際のゲーム「セカンドワンナイト人狼」の仕様と食い違っている。これを実装準拠に直し、役職カード画像を使って ROLES セクションをより魅力的にする。**デザイン（レイアウト・配色・アニメーション・SCSSトークン）は維持し、データ定数とコピー文の差し替え + 役職画像の追加が中心。**

## 現状の食い違い（実装との差分）

実装は「自分で役職を選べる」ワンナイト人狼の変種（`frontend/src/features/werewolf/components/rule.tsx`、`backend/.../entity/werewolf/roll/`、`WereWolfConst.java` で確認）。

- **遊び方**: LPは古典ワンナイトの4手順（配る/眠る/話す/指さす）。実際は **役職選択 → 議論 → 投票** の3手順で、**夜フェーズは存在しない**（能力は議論中にオープンに使う）。NPCが必ず1人加わる点、スタートPに2枚配られ選んで残りを回す点が未記載。
- **役職**: LPの「占い師=中央2枚を覗く」「怪盗=誰かと入替」「吊人」は古典の内容で誤り。実装の役職と能力・陣営が違う。
- **プレイ人数**: LPは「3〜8人」。正しくは **３人〜**（作者確認済み）。

## 変更方針

`src/pages/index.tsx` の定数（`STATS` / `STEPS` / `ROLES`）とコピー文を差し替え、`src/styles/lp.module.scss` に役職画像用の `.roleImg` を1ブロック追加。レイアウト構造・既存の配色/hover演出は流用する。

### 世界観コピーの扱い

タイトル・`紅葉`・`二度目の夜へ` 等の詩的表現は **雰囲気として維持**。ただし「夜フェーズがある」と読める“ルール断定”の文（例: ABOUTの「全員が目を閉じる夜を越えたら」）だけ書き換える。

### ROLESカード構造の変更

既存 `.roleCard` の中身を差し替える:

```
<div class={roleCard + camp}>
  <img class={roleImg} src={/images/werewolf/roll/${rollNo}.jpg} alt={name} loading="lazy" />
  <div class={roleHead}>
    <h3>{name}</h3>
    <span class={roleCamp}>{campLabel}</span>
  </div>
  <p>{text}</p>
</div>
```

- `.roleGlyph` は使用しなくなる（SCSS定義は残しても害はないが、未使用なら削除可）
- `.roleHead` は既存の flex/space-between を流用（左=役職名 h3、右=陣営バッジ）。h3 のマージン調整のみ必要になる場合あり
- `.roleImg` を新規追加: `width:100%; aspect-ratio:1/1; object-fit:cover; border-radius:12px; display:block;`
